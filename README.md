ProgressiveParts
ProgressiveParts — это мод для Kerbal Space Program, который добавляет многоуровневые копии существующих деталей с прогрессивными характеристиками. Каждая деталь получает улучшенные версии (Tier 1–7), которые:

Становятся доступны по мере продвижения по технологическому дереву.

Имеют повышенную массу, ёмкость батарей, дальность антенн, научную ценность и другие параметры.

Требуют от игрока учитывать давление, температуру и радиацию благодаря интеграции с модом Kerbalism.

🚀 Возможности
7 уровней улучшений (Tier 1 … Tier 7) для каждого поддерживаемого типа деталей.

Автоматическая привязка к технологическому дереву — копии открываются в узлах, соответствующих их Tier (совместимо со стандартным TechTree.cfg или Community Tech Tree).

Поддержка разных категорий деталей:

Аккумуляторы (ProgressiveParts_Battery)

Командные модули / зонды (ProgressiveParts_Command, ProgressiveParts_Probe)

Теплозащита (ProgressiveParts_Heatshield)

Освещение (ProgressiveParts_Light)

РИТЭГи (ProgressiveParts_RTG)

Научные приборы (ProgressiveParts_Science)

Колёса / движители (ProgressiveParts_Wheel)

Модификации характеристик для каждого уровня:

Масса: 1.0 → 2.2 (шаг ~0.2 за уровень)

Ёмкость батарей: 1.0 → 4.3

Дальность антенн: 1.0 → 8.0

Научная ценность и коэффициент передачи: 1.0 → 4.9 и 1.0 → 3.2

Скорость и диапазон актуаторов (гирборды, роботика)

Мощность и дальность света

Сопротивление парашютов и скорость раскрытия

Требования к окружающей среде (давление, температура, радиация) через модуль Reliability мода Kerbalism. Значения прогрессивно возрастают с уровнем.

📦 Состав
Мод поставляется в виде набора конфигурационных файлов, каждый из которых отвечает за определённую категорию деталей:

ProgressiveParts_Battery.cfg — копии аккумуляторов.

ProgressiveParts_Command.cfg — копии командных модулей.

ProgressiveParts_Heatshield.cfg — копии теплозащиты.

ProgressiveParts_Light.cfg — копии источников света.

ProgressiveParts_Probe.cfg — замена зондов и процессоров.

ProgressiveParts_RTG.cfg — копии РИТЭГов.

ProgressiveParts_Science.cfg — копии научных приборов.

ProgressiveParts_Wheel.cfg — копии колёс и створок.

Каждый файл генерируется специальным Python-скриптом (включён в репозиторий), который сканирует вашу установку KSP и создаёт копии всех подходящих деталей из установленных модов.

🛠️ Установка
Установите моду:

Kerbalism

ModuleManager 

Community Tech Tree (опционально, для расширенного дерева технологий)

Скачайте последний релиз ProgressiveParts или используйте генератор скриптов.

Поместите содержимое архива в папку GameData.

Убедитесь, что файлы ProgressiveParts_*.cfg находятся в любой подпапке GameData (например, GameData/ProgressiveParts).

⚙️ Генерация собственных настроек
Если вы хотите изменить количество уровней, множители или характеристики, используйте поставляемый Python-скрипт:

bash
python generate_progressive_parts.py
Скрипт запросит путь к GameData, найдёт все детали по ключевым словам и создаст актуальные .cfg файлы. Вы можете отредактировать параметры в начале скрипта:

NUM_TIERS = 7 — количество уровней.

TIER_DATA — таблица давления, температуры, радиации для каждого уровня.

LEVEL_MODIFIERS — множители массы, ёмкости, дальности и т.д.

🎮 Игровой процесс
С каждым новым уровнем:

Масса детали увеличивается (улучшенное оборудование тяжелее).

Эффективность растёт: батареи держат больше заряда, антенны передают дальше, научные приборы дают больше очков.

Требования к окружающей среде ужесточаются: детали высокого уровня выходят из строя при экстремальных давлениях или температурах, если не обеспечить защиту.

Это побуждает игрока продумывать конструкцию кораблей и использовать экранирование, теплозащиту и радиационную защиту, что особенно актуально в реалистичных модах (KSRSS, RSS/RO).

🔧 Совместимость
Kerbalism — требуется для работы модуля Reliability.

Community Tech Tree — поддерживается, узлы автоматически назначаются на основе стоимости.

Стандартное технологическое дерево — работает из коробки (использует TechTree.cfg).

Другие моды: скрипт поиска распознаёт детали из модулей DMagic Orbital Science, Station Science Continued, SCANsat, Real Antennas, Near Future Exploration и многих других.
